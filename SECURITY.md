# Security Policy

## Supported Use

This project is a learning-focused Wi-Fi security lab.  
It is designed for:
- Practicing authorized wireless security testing
- Understanding how to better defend personal Wi-Fi networks
- Learning how offline attacks against weak passwords work

## Reporting a Vulnerability

If you discover a security issue related to this repository (for example,
an unsafe example or missing warning), please:

1. Avoid posting sensitive details publicly.
2. Open a GitHub issue with a high-level description of the problem, **OR**
3. Contact the author privately via GitHub.

Do NOT use this project to attack any network without explicit permission.

## Legal & Ethical Requirements

- Only test networks that you fully control or have written authorization for.
- Respect local laws and regulations.
- This project must not be used for malicious or illegal activity.
